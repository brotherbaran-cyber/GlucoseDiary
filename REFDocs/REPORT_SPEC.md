# Report Specification

## Report Type
Printable 30-day summary PDF

## Header
- App name
- Patient/caregiver label if later desired
- Report generation date
- Covered period (start date to end date)

## Summary Section
- Total number of entries
- Average CGM reading
- Average manual reading
- Average difference
- Highest and lowest reading values
- Counts by context tag

## Detailed Table Columns
- Date
- Time
- Context
- CGM
- Manual
- Difference
- Note

## Page Rules
- Repeat table headers on each page if multi-page
- Use landscape mode if needed for width
- Keep numeric columns aligned

## Export Behavior
- Generate PDF to app cache or user-selected location
- Allow Android print/share flow
