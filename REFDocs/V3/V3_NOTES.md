# V3 Notes

## Intent
This V3 bundle is meant to move the app closer to day-to-day real use for a caregiver or spouse.

## Functional coverage
- paired readings for CGM and manual blood draw
- date and time per entry
- context classification
- rolling 30-day summary
- month-by-month summary
- PDF export
- CSV export
- Android file sharing
- basic daily reminder scheduling scaffold

## Notable UX improvements
- date and time can now be selected with dialogs instead of only being typed manually
- large CGM/manual differences can be highlighted
- export actions are separated so the user knows whether a PDF or CSV is being produced
- Settings tab centralizes reminder and export utilities

## Remaining production polish ideas
- backup to cloud
- passcode / privacy layer
- stronger tests
- print preview polish on different OEM devices
- sorting and filtering in history
- doctor-facing report layout with branding
